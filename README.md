# Fitness Tracking Web Application

A full-stack Fitness Tracking Web Application developed as part of our MCA Mini Project.  
The application helps users monitor and improve their fitness journey by tracking workouts, calorie intake, BMI, fitness goals, and overall health progress through an interactive dashboard.

---

# Project Overview

This web application is designed to provide users with an easy and efficient way to manage fitness-related activities digitally. Users can log workouts, track calories burned and consumed, monitor weekly goals, calculate BMI, and generate progress reports.

The system also includes an Admin Panel for managing users and monitoring application data.

#  Challenges Faced

Initially, the backend was developed using Strapi CMS for rapid API and content management development. However, as the project expanded and handled more features and data, responsiveness and backend performance issues were observed.

To overcome these challenges, the backend architecture was migrated from Strapi to MongoDB with a custom Express.js backend implementation.

This migration helped improve:
- Performance
- Scalability
- Flexibility
- Database Control

The transition also provided valuable hands-on experience in backend restructuring, database management, and debugging.
